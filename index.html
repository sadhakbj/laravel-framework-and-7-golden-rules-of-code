### Creator of Laravel --- Taylor Otwell ![taylor](images/taylor.png "taylor")
### 7 Reasons to use Laravel Framework --- * Excellent Documentation. * Laracasts * Intuitive Syntax * Artisan Code Generation * Out-of-the-box User Model * Blade Templating Engine * Supporting Products and Packages
### Reason 1 : Excellent Documentation --- * Easy to start up with laravel. * Visit [Laravel Docs](http://www.laravel.com/docs)
### Reason 2 : Laracasts --- * Laravel is supported by Laracasts. * Discussion Forum * Features a mix of free and paid videos that show you how to use Laravel * Laracasts is addicting * Owned by Jeffrey Way
### Jeffrey Way --- * An expert and well-versed instructor ![jeffrey](images/jeff_way.jpeg "jeffrey")
### Reason 3 : Intuitive Syntax --- * Follows principle of clear and maintainable code * Examples: ``` // asking Role models for all the roles with pagination. $roles = Role::paginate(5); ``` Another ``` // asking for single model instance , throw exception if not found $role = Role::findOrFail($id); ```
### Reason 4: Artisan Code Generation --- * Laravel ships great command line tool named Artisan * provides a number of helpful commands * Example: Make a new Controller ``` php artisan make:controller SampleController ``` * Example: Make a new Migration ``` php artisan make:migration create_articles_table --create ```
### Reason 5: Out-of-the-box User Model --- * Inbuilt user model with controllers and migration to setup user table * Fully working user registration and login system, with forgot password functionality
### Reason 6: Blade Templating Engine --- * Simple, yet powerful templating engine provided with Laravel * Makes working with the typical PHP/HTML much better * Typical PHP / HTML : ``` if(accepted == true) { echo $userName; } ``` * Blade templating engine: ``` @if(accepted == true) {{ $userName }} @endif ```
### Reason 7: Supporting Products and Packages --- * Well-supported by many packages * Services from Laravel: * Lumen * Homestead * Elixir * Forge * Help from beginning from local environment to deployment
